# Credit Card Financial Dashboard

A Power BI–based financial analytics dashboard designed to analyze credit card transactions, customer behavior, and revenue trends with a strong focus on weekly and year-to-date performance insights.

---

## Project Overview

This project focuses on building an interactive **Credit Card Financial Dashboard** using **Power BI**, supported by **SQL** and **DAX**.  
The dashboard enables stakeholders to track key financial metrics, monitor customer usage patterns, and evaluate business performance across time, card categories, and demographics.

The primary goal is to transform raw financial and customer data into **clear, actionable insights** through effective data modeling and visualization.

---

## Objective

- Analyze credit card transaction and customer data  
- Monitor weekly and year-to-date revenue trends  
- Understand customer demographics and spending behavior  
- Identify high-performing card categories and regions  
- Support data-driven business decisions through visualization  

---

## Dataset Description

The project uses structured financial datasets containing:

- Credit card transaction data  
- Customer demographic information  
- Weekly time-based records  
- Revenue components such as transaction amount, interest earned, and annual fees  

Data was stored and processed using **SQL**, then connected to **Power BI** for analysis and visualization.

---

## Tools & Technologies

- **Power BI** – Dashboard creation and data visualization  
- **SQL** – Data storage, querying, and transformation  
- **DAX** – Measures and calculated columns  
- **CSV** – Source data files  

---

## Project Workflow

1. Collected and prepared raw CSV datasets  
2. Created tables and loaded data into SQL  
3. Connected SQL database to Power BI  
4. Cleaned and modeled data relationships  
5. Created DAX measures and calculated fields  
6. Designed an interactive and visually consistent dashboard  
7. Analyzed weekly and cumulative performance metrics  

---

## Key Insights

### Week-on-Week (WoW) Performance
- Revenue increased by approximately **28.8%**
- Growth observed in total transaction amount and transaction count
- Increase in the number of active customers
- Indicates improved customer engagement and spending behavior

### Year-to-Date (YTD) Performance
- **Total Revenue:** 57M
- **Total Interest Earned:** 8M
- **Total Transaction Amount:** 46M

### Customer & Card Insights
- Male customers contribute higher revenue compared to female customers
- Blue and Silver credit cards account for nearly **93% of total transactions**
- Mid-income and working-age customer segments show the highest activity

### Regional Insights
- Texas, New York, and California together contribute approximately **68% of total revenue**
- These states consistently show high transaction volumes

### Operational Metrics
- **Activation Rate:** 57.5%
- **Delinquency Rate:** 6.06%
- Indicates strong customer onboarding and controlled credit risk
